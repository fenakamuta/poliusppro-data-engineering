## Como começar (edição 2026)

```bash
git clone https://github.com/fenakamuta/poliusppro-data-engineering
cd poliusppro-data-engineering/2026
pip install -r requirements.txt
```

Comece pela pasta [`2026/00-setup/`](./2026/00-setup/) para preparar o ambiente, depois siga para a [`aula-1-dados-na-pratica/`](./2026/aula-1-dados-na-pratica/).

## Filosofia do curso 2026

- **Open-source e gratuito** — nada que dependa de cartão de crédito ou nuvem paga.
- **Hands-on desde o primeiro minuto** — você roda código de verdade, com dados reais.
- **IA aplicada como ferramenta** — usamos Cursor, LLMs e agentes para acelerar.
- **Tudo reproduzível em casa** — qualquer máquina com Python 3.10+ funciona.
